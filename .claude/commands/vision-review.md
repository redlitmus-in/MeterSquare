# Vision-Based UI/UX Review

Using Claude's vision capabilities (multimodal cookbook pattern), analyze the provided image or screenshot.

## Review Areas

### 1. Visual Design
- Layout and spacing consistency
- Color scheme and contrast
- Typography hierarchy
- Visual balance and alignment

### 2. User Experience
- Navigation clarity
- Call-to-action prominence
- User flow intuitiveness
- Error message visibility

### 3. Accessibility
- Color contrast ratios (WCAG compliance)
- Text readability
- Interactive element sizes
- Alt text and labels

### 4. Responsiveness
- Mobile-friendly elements
- Breakpoint considerations
- Touch target sizes

### 5. Best Practices
- Comparison to modern design standards
- Industry-specific conventions
- Usability heuristics

## Output

Provide:
- Screenshot analysis with specific observations
- Actionable recommendations with priority
- Code suggestions for improvements
- Before/after mockup descriptions if applicable

**Attach image with:** `@path/to/screenshot.png`
